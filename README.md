# Dark Blend – Automated Vulnerability Scanner

## Description

Dark Blend is a Python-based mini project developed using Flask.
It is a web-based cybersecurity tool that automatically scans systems, networks, and websites to detect security vulnerabilities in a simple and user-friendly way.

---

## Features

* Secure login with email and OTP verification
* Web scanning (security headers, SSL, DNS, subdomains)
* Network/IP scanning using Nmap (open ports, risky services)
* System scanning (CPU, memory, disk, processes)
* Dashboard with vulnerability severity (Critical / High / Medium / Low)
* AI-based suggestions to fix detected issues
* Report generation

---

## How to Run
### Install requirements
```bash
pip install -r requirements.txt
```
### Run the application
```bash
python app.py
```

---

## Tech Stack
* Python
* Flask
* PostgreSQL

## Conclusion
Dark Blend simplifies vulnerability scanning by combining multiple security checks into one platform, making cybersecurity easier and more accessible.
